# 工艺术语归一与排查速查卡

## 1. 术语归一

- 虚焊 = 假焊 = 焊接不实
- 焊盘发暗 = 焊盘氧化风险
- 起泡 = 鼓包（在贴合场景下）
- 分层 = 层间结合力不足
- 对位偏移 = 图形偏位

## 2. 焊接类问题速查

当出现“虚焊 + 焊盘发暗 + 夜班湿度偏高”时，建议按以下顺序处理：

1. 更换新助焊剂并做对比试焊
2. 清洁或复活焊盘表面
3. 复测焊头实际温度稳定性
4. 抽查下压压力与夹具一致性

## 3. 印刷与固化联动速查

当出现“印刷层偏薄 + 固化波动 + 电阻偏高”时，不要只看单工序：

1. 同步回看印刷厚度记录与固化曲线
2. 验证批内阻值波动和附着力变化
3. 检查环境湿度与浆料状态

## 4. 贴合与耐压联动速查

当出现“起泡 + 耐压不良”时，建议优先回看：

1. 材料吸湿状态与暴露时间
2. 表面清洁执行情况
3. 压合温度与保压时间
4. 压力建立方式

## 5. 何时升级工程师

满足任一条件建议升级：

- 同批连续重复异常且趋势上升
- 跨工序联动异常（如印刷-固化、贴合-耐压、模压-翘曲）
- 已做单变量验证仍无法定位
