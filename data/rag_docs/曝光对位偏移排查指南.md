# 曝光对位偏移排查指南

## 1. 现象说明

曝光对位偏移通常表现为图形边界错位、局部重影、后续显影后图形偏离基准位置。轻微偏移在前段可能只表现为外观差异，到了后段电测或装配时才显现影响。

## 2. 优先排查方向

### 2.1 设备定位系统

定位相机、基准识别精度、平台重复定位能力是首要排查对象。若同一批次中偏移方向一致，优先怀疑设备坐标补偿或定位基准漂移。

### 2.2 基材尺寸变化

柔性材料受温湿度和前段热历史影响，会出现轻微伸缩。若前道烘烤后未充分回稳就进入曝光，对位偏差更容易放大。

### 2.3 治具与夹持

夹持不平、边缘翘起或定位销磨损，都可能让局部区域偏移明显大于整体平均值。此类问题常见于小批多换型场景。

## 3. 典型排查步骤

1. 先确认偏移是整体一致还是局部集中
2. 检查最近一次设备校正记录
3. 回看前段温湿度和热处理历史
4. 抽查治具平整度、定位销磨损和夹持稳定性
5. 用同一批材料做复测，避免批次差异干扰判断

## 4. 容易混淆的情况

### 4.1 不是真偏移，而是前段图形边缘异常

若印刷或前段图形本身边界不完整，现场可能误以为是曝光偏移。此时应先确认基准图形是否完整。

### 4.2 局部偏移与基材回弹混在一起

模压、贴合或烘烤后的残余应力，可能让材料在曝光前后形变不一致。这类问题单靠设备微调往往解决不了。

## 5. 现场建议

- 班组先记录偏移方向、范围和批次一致性
- 若连续 3 批出现同向偏移，应升级给工程师检查设备补偿和前段热历史
- 对高湿环境下的薄基材，曝光前应确保尺寸稳定时间充足

## 6. 结论

曝光对位偏移既可能来自设备，也可能来自材料状态。若只盯着设备参数，很容易漏掉前段烘烤、贴合或存放条件带来的尺寸变化问题。
