# Security Policy

## Supported Scope

This repository is currently a prototype. Security fixes are accepted on the latest default branch.

## Reporting a Vulnerability

Please do **not** open a public issue for sensitive vulnerabilities.

Send a private report with:
- A clear description of the issue
- Reproduction steps
- Potential impact
- Suggested fix (if available)

Temporary contact channel: open a private communication path with the maintainer before disclosure.

## Response Expectations

- Initial acknowledgement: within 7 days
- Triage result: within 14 days when reproducible
- Fix timeline: based on severity and complexity
